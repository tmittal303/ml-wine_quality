# Wine Quality Prediction — Machine Learning Project

ML project that predicts whether a wine is **GOOD** or **BAD** 
based on its properties.

---

## Project Overview
- **Dataset:** Wine Quality Dataset 
- **Type:** Binary Classification (GOOD = quality ≥ 7, BAD = quality < 7)
- **Tool:** Python, Google Colab / VS Code

---

## Steps Covered
1. Data loading and basic analysis (`head`, `info`, `describe`)
2. Missing value check and correlation analysis
3. Binary target variable engineering
4. Train-test split 
5. Logistic Regression (without scaling)
6. StandardScaler — before vs after comparison
7. Model comparison: Logistic Regression vs KNN vs Decision Tree
8. Hyperparameter tuning with GridSearchCV
9. Feature importance analysis
